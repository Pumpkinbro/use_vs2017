## example3

本例子展示如何制作一个 vcpkg 包。


## 代码

本例子代码来至于 https://github.com/zieckey/vcpkgdemo


## 相关命令

比如：

```shell
vcpkg create vcpkgdemo https://github.com/zieckey/vcpkgdemo/archive/1.0.zip vcpkgdemo-1.0.zip
```

```shell
vcpkg build vcpkgdemo
```

```shell
vcpkg install vcpkgdemo
```
